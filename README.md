## 주요기능
- Firebase 로그인, 회원가입
- 상품 목록(Pagination)
- 장바구니
- 찜 리스트
- 카테고리 검색(Pagination)
- 마이페이지
- 결제페이지(실제 결제x)
- 주문목록
- 관리자 페이지(Pagination)(상품 추가, 수정, 삭제)

## Getting Started

First, run the development server:

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
